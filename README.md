# FakeSocialNetwork

Social network type Android application that uses a local database implemented with Room.

## Features

- Core
- Login
- Register
- Profile
- Add
- Search
- Profile
- Post Edit
- Profile Edit

## Tech Stack

- Coil
- XML
- Kotlin
- Room
- MVVM
- LiveData
- Retrofit2
- StateFlow
- Coroutines
- Navigation
- ViewModel
- Pretty Time
- Dagger Hilt
- ViewBinding
- Image Picker
- Android Studio
- SwipeRefreshLayout
